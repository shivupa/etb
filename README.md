# Even tempered basis generator
Requires PySCF

```
usage: etb.py [-h] [-i INPUT] [-exi EXINPUT]

options:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        input file. First line atom, second line-nth line : l, n, alpha, beta.
  -exi EXINPUT, --exinput EXINPUT
                        example input file
```
